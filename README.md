## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm dev

# build
$ pnpm build

# run build output
$ pnpm start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
